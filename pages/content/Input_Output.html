Input Output

This theme is compose of BIF, BIFXML, DSL, NET, UAI, XDSL

The Input/Output task involves reading and writing Bayesian Network files.

Bayesian Network files are used to store the structure and parameters of a Bayesian Network.

They can be used to save a Bayesian Network for later use, or to share a Bayesian Network with others.

They can also be used to import a Bayesian Network into another program, such as a Bayesian Network editor or a Bayesian Network inference engine.
